2022 旅游计划

新疆（北疆）

计划时间：

2022/10/01 ~ 2022/10/07（国庆期间）

需要准备的东西：

  1. 衣物（御寒 + 防晒）-> 深秋 + 冬装

    • 秋衣 + 秋裤 + 长袜（早晚很冷）✅
    • 三合一冲锋衣/薄羽绒服 🈚🧥
    • 围脖 ✅
    • 长裙 + 暖色系衣服（拍照）✅
    • 马丁靴（骑马、上山、走泥地）🈚👢
    • 防晒衣 + 防晒霜 + 墨镜 + 帽子 + 遮阳伞 🈚🧢

  2. 证件类

    • 身份证
    • 驾驶证
    • 学生证/军人证 等可以优惠的证件

  3. 洗漱用品（📢保湿保湿保湿）

    • 牙刷 + 牙膏（旅行款）
    • 洗面奶 + 面霜 + 补水面膜 + 擦脸精华 🙎‍♀️
    • 一次性面巾 + 浴巾
    • 身体乳（手脚涂）
    • 分装瓶 + 贴纸 ⭐（可以装沐浴露/洗发露/护发素啥的）
    • 润唇膏
    • 一次性内裤（可选：一次性马桶坐垫/一次性泡澡袋）

  4. 生活用品

    • 一次性袋子 🈚
    • 暖宝宝 🈚
    • 姨妈巾
    • 口罩
    • 纸巾 + 湿巾
    • 刮胡刀
    • 指甲钳
    • U型枕（车内睡觉）
    • 保温杯
    • 蒸汽眼罩（休息）
    • 首饰收纳盒 ⭐
    • 少量现金

  5. 药物类

    • 999 感冒药 + 退烧药
    • 肠胃药
    • 晕车药
    • 止泻药（藿香正气）
    • 创可贴
    • 消毒酒精
    • 布洛芬

  6. 电子产品类

    • 手机、充电器、数据线（USB + Type-C）、充电宝（多带几个容量大的）、耳机
    • 相机（电池、充电器、三脚架、内存卡）
    • 电源转换插头

  7. 化妆品类

    • 保湿喷雾
    • 底妆
    • 遮瑕
    • 眉笔
    • 眼影
    • 睫毛膏
    • 口红腮红
    • 卸妆水
    • 卸妆棉
    • 美妆蛋

  8. 食品类

    • 自嗨锅/自热饭
    • 泡面 + 面包
    • 零食（车上可以吃）
    • 饮料（不用准备太多，可以到那里补给）

需要了解的东西：

  1. 防疫政策
  2. 需要注意的民族习俗（比如清真）
  3. 可能有哪些坑

待买：

  1. 衣物

    • 三合一冲锋衣
    • 马丁靴
    • 帽子

  2. 洗漱用品

    • 分装瓶 + 贴纸
    • 一次性内裤

  3. 生活用品

    • 一次性袋子（装零散东西、垃圾）—— 公共
    • 暖宝宝 —— 公共
    • U型枕 + 蒸汽眼罩（车内睡觉）——公共

  4. 药物类

    • 999 感冒药 + 退烧药
    • 肠胃药
    • 晕车药
    • 止泻药（藿香正气）

  5. 电子产品

    • 电源转换插头
    • 充电宝 4w毫安
